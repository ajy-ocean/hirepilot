    <strong>HirePilot</strong> is a sleek, minimalistic, high-performance mobile application built to bridge the structural gap between competitive developer resumes and modern Applicant Tracking Systems (ATS). Powered by Google's advanced <strong>Gemini 2.5 Flash API</strong>, it delivers instant compliance grading, gap visualization matrix trackers, and dynamically tailored mock technical interview simulators.

## ✨ Primary Capabilities

- **Automated Text Ingestion System:** Extends seamless raw text extraction logic over `.txt` and complex multi-column `.pdf` document files.
- **ATS Semantic Alignment:** Cross-references resume context targets against user-pasted Job Descriptions to calculate simulated alignment metrics.
- **Competency Gap Matrix:** Exposes critical missing technical frameworks and identifies optimization points.
- **Dynamic Interview Simulator:** Produces contextual interview questions mapping directly to detected missing skills, complete with recruiter evaluation rationales.
- **Adaptive UI Platform:** Features highly scannable minimalistic dashboard designs with system-wide native Light and Dark mode options.

---

## 🛠️ Technology Stack Architecture

- **Frontend Core Framework:** Flutter (Dart) using highly-scalable `ChangeNotifier` state distribution vectors.
- **AI Core Processing Engine:** Google Gemini 2.5 Flash (via official `google_generative_ai` software development bundle hooks).
- **Document Parsing Framework:** Syncfusion High-Performance PDF Text Extraction pipeline.
- **State Management Pattern:** Provider Architecture layer.
